# Student Management System（学生管理系统）



## 项目简介

Java后端实习项目，迭代开发中：从控制台版 → Spring Boot Web版 → Redis优化版。目标：暑期拿到杭州/大厂Java实习（月薪8-15k）。



## 技术栈

- Java 17 + Maven

- Spring Boot（即将升级）

- MySQL + MyBatis

- GitHub: https://github.com/liver-m/StudentManagementSystem
